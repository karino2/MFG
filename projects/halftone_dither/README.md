# Halftone Dither

Without antialiasing, pixels are strictly black or white. With antialiasing enabled, shades of gray are used to smooth the edges.

Antialiasing and angle only apply to "square" and "sin" shape types.

Dither pattern is based on following.

- Newspaper: Bayer Pattern
- Interleaved: [Jorge Jimenez – Game Developer – Next Generation Post Processing in Call of Duty: Advanced Warfare](https://www.iryoku.com/next-generation-post-processing-in-call-of-duty-advanced-warfare/)
